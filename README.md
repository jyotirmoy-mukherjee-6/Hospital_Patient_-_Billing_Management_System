# Hospital_Patient_-_Billing_Management_System

Hospital Patient & Billing Management System
A compact Excel-based hospital management project designed to manage patient records, billing operations, and dashboard analytics using only Microsoft Excel features.
Features
•	Patient registration and record management
•	Billing and payment tracking
•	Interactive dashboards and visual reports
•	Automated calculations and summaries
•	Data cleaning and preprocessing
•	Macro-enabled automation
•	User-friendly interface

Excel Tools & Techniques Used
Data Cleaning
•	Remove Duplicates
•	Data Validation
•	Handling Missing Values

Excel Functions & Features
•	VLOOKUP
•	IF Statements
•	SUM / SUMIF
•	COUNT / COUNTIF
•	Pivot Tables
•	Pivot Charts
•	Conditional Formatting
•	Drop-down List
•	Excel Macros 
•	Count Blank


Dashboard Components
•	Dynamic Charts
•	Patient & Revenue Analysis
•	Billing Summary Reports


Purpose
This project demonstrates how Microsoft Excel can be used to build a lightweight hospital management and billing solution with analytics and automation features.
